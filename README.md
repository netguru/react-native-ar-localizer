# ar-localizer-rn

React Native bridge for ARLocalizer

## Installation

```sh
npm install ar-localizer-rn
```

## Usage

```js
import ArLocalizerRn from "ar-localizer-rn";

// ...

const result = await ArLocalizerRn.multiply(3, 7);
```
